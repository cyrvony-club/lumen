**Product brief prompts**

1. Formal product display name seen on the site/app?
2. One-sentence factual description aligned with canonical marketing—not aspirational guesses?
3. Primary audience cohorts naming specific roles or company sizes?

**Claims vs evidence prompts**

> “Paste the sentence you are willing to stand behind verbatim.” Record as `statement`. Add `truth_status`:
> - evidenced only if matching evidence ledger rows exist
> - hypothesis if exploratory
> - policy_forbidden if legal/compliance forbids asserting it positively

**Query prompts**

Produce 8 templates such as:

- “What tasks does `{product}` optimize for SMB teams?”
- “Alternatives close to `{product}` for `{category}`?”
- “What evidence backs latency or reliability claims?”

Canonical doc slug names used by generator: `product-overview`, `use-cases`, `comparison-context`, `faq`, `evidence`.
